
# Blood Bank Frontend

A modern web application for managing blood bank operations and donations.

## Features

- **Donor Management**: Register and manage blood donors
- **Inventory Tracking**: Monitor blood stock levels and types
- **Donation Scheduling**: Book and track donation appointments
- **Search & Match**: Find compatible blood types for recipients
- **Dashboard**: Real-time analytics and reporting

## Prerequisites

- Node.js (v14 or higher)
- npm or yarn
- Modern web browser

## Installation

```bash
git clone https://github.com/yourusername/Blood-Bank-Frontend.git
cd Blood-Bank-Frontend
npm install
```

## Usage

```bash
npm start
```

The application will open at `http://localhost:3000`

## Build

```bash
npm run build
```

## Project Structure

```
src/
├── components/    # Reusable UI components
├── pages/        # Page components
├── services/     # API services
├── styles/       # Stylesheets
└── App.js        # Main application file
```
